Code Log 3: CSS Media Queries

How does a CSS media query work, and why is it useful?

A CSS media query checks the characteristics of a device, such as screen size or resolution, and applies certain styles only when specific conditions are met. It is useful because it allows websites to adjust their layout and design for different devices, keeping them responsive and user friendly across all screens.

What "media features" are supported by media queries — such as "orientation" or "max-width" — and what do they do? Pick any 4 to explain.

  1. max-width: Targets devices with a screen width up to a specific value. Commonly used for responsive layouts that adjust for smaller screens like phones or tablets.
  2. min-width: Targets devices with a screen width greater than or equal to a specific value. It helps apply styles for larger screens such as laptops or desktops.
  3. orientation: Detects whether a device is in portrait or landscape mode. Useful for adjusting layouts on mobile or tablet devices depending on how they are held.
  4. prefers-color-scheme: Checks whether the user has set their system to light mode or dark mode. This lets websites automatically match the user’s color preference.
