How CSS Uses Color

CSS (Cascading Style Sheets) controls the color of text, backgrounds, and borders on a webpage. You can apply color to nearly any visual element by using specific CSS properties.

Key CSS Color Properties

  • color: Sets the color of text. Example: color: darkblue;
  • background-color: Sets the fill color behind an element. Example: background-color: lightyellow;
  • border-color: Sets the color of an elementâ€™s border. Example: border-color: rgb(255, 99, 71);

Ways to Specify Color in CSS

CSS accepts several different color formats:

  1. Named Colors: Use simple names that browsers recognize, like red, blue, or darkolivegreen.
    Example: color: red;
  2. Hexadecimal: A six-digit code starting with # that combines red, green, and blue values.
    Example: color: #FF6347; (a shade of tomato red)
  3. RGB: Uses numbers between 0 and 255 for red, green, and blue components.
    Example: color: rgb(255, 99, 71);
  4. RGBA: Like RGB, but includes an alpha channel (transparency) from 0.0 to 1.0.
    Example: background-color: rgba(255, 99, 71, 0.5);
  5. HSL/HSLA: Uses hue, saturation, and lightness values, with optional alpha for transparency.
    Example: color: hsl(120, 60%, 40%); (a medium green)

Using these different formats gives designers flexibility to create consistent color schemes across a webpage.
