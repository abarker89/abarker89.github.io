How CSS Uses Color

CSS controls the color of text, backgrounds, and borders on a webpage. You can apply color to nearly any visual element by using specific CSS properties.

Key CSS Color Properties

  • color: Sets the color of text. Example: color: darkblue;
  • background-color: Sets the fill color behind an element. Example: background-color: lightyellow;
  • border-color: Sets the color of an elementâ€™s border. Example: border-color: rgb(255, 99, 71);

Ways to Specify Color in CSS

  1. Named Colors: color: red;
  2. Hexadecimal: color: #FF6347;
  3. RGB: color: rgb(255, 99, 71);
  4. RGBA: background-color: rgba(255, 99, 71, 0.5);
  5. HSL: color: hsl(120, 60%, 40%);
