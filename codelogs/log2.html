Code Log 2: CSS Measuring Units

Absolute vs. Relative Units

Absolute Units

  • Examples: px, cm, in
  • Fixed sizes that don’t change with context.

Relative Units

  • Examples: em, %, vw
  • Scale based on parent font size, the root, or the viewport.
  • Flexible for responsive design.

Three Unit Sets

em, rem

  • em: relative to the parent’s font size (2em = 32px if parent is 16px).
  • rem: relative to the root font size, unaffected by nesting.
  • Use: rem for consistent scaling, em for local adjustments.

vw, vh

  • vw: 1% of the viewport width.
  • vh: 1% of the viewport height.
  • Use: good for full-screen sections (height: 100vh).

%

  • Relative to the parent element’s size (width, height, or font).
  • Use: fluid, responsive layouts (e.g., width: 50% of parent).

A bonus benefit is that semantic tags also make code more readable and maintainable for developers by clearly showing what each section of content represents.
